# Flight app

This is an application that gives an abillity to easily find and save flight tickets account.

## Implemented things: 
* Registration and login using Bearer Token
* Routing
* Crud operations 
* Hooks
* Sending and updating data to the server in real time
* Working with forms
* Validation



## Technologies:
* ReactJS
* NodeJS
* Express
* PostgreSQL
